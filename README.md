# OracleSQL-MiniProject
These project is about improvement of queries with Oracle SQL.
AS A RESULT
Firstly tables created and our data uploaded like 700k data. 
Secondly some queries wrote down, their excecution times and execution plans are checked
And then some improvements started applying on these queries to improve their execution times.
One of them Indexes. This one is most effective improvement technique on the queries. 
The other one is Partitions. 
Last one is also effective on some queries which contains some calculations or string search.
